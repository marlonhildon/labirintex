Jogo usado como prática para alguns conceitos de IA, que foram:

* Árvores de decisão;
* Algoritmo A* (A Star) através do NavMesh da engine.

Para tal, o framework **Accord** (para machine learning) foi usado.

Caso queira testar o jogo compilado, acesse a pasta **Executable**.
